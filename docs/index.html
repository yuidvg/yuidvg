概要

2001年大阪府出身。 幼少期から科学に興味を持ち、幼稚園生の頃から研究所に通っていた。アートにも興味を持ち、小学6年生のアートギャラリーでの個展や、ニコンプラザ大阪での写真展を弟と共同で開催した。 中学2年の夏、全国から理科好きの中学生40人を選抜してノーベル賞受賞者などの講義を受ける創造性の育成塾に選抜される。大阪府立茨木高等学校に入学。未踏ジュニアのスーパークリエイター認定・LINE BOOT AWARDS 2018グランプリ受賞をきっかけにN高ネットコースへ。 SXSW Edu Student Startup CompetitionのFinalist。筑波大学情報メディア創生学類に入学。2023年から 42 Tokyoに入学。現在は42 Parisに在学。 登山と歴史に興味がある。
